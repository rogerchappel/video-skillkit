# Security Policy

Please do not include secrets, private repository data, unreleased product material, or sensitive customer examples in public issues.

For security-sensitive reports, contact the maintainer privately before publishing details. Include the affected version or commit, the command you ran, and the smallest safe reproduction you can share.

`video-skillkit` is local-first: it reads local repository files and writes local brief artifacts. Report any behavior that unexpectedly executes code, uploads files, or contacts external services.
